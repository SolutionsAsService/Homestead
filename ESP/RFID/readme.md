
                   ## Wiring Diagram
                    
                    +-----------------------------+
                    |          ESP32              |
                    |                             |
                    | 3.3V  ---------------------> OLED VCC  
                    | GND   ---------------------> OLED GND  
                    | GPIO21 --------------------> OLED SDA  
                    | GPIO22 --------------------> OLED SCL  
                    |                             |
                    | 3.3V  ---------------------> RFID 3.3V  
                    | GND   ---------------------> RFID GND  
                    | GPIO5 ---------------------> RFID SDA (CS)  
                    | GPIO18 --------------------> RFID SCK  
                    | GPIO23 --------------------> RFID MOSI  
                    | GPIO19 --------------------> RFID MISO  
                    | GPIO2  --------------------> RFID RST  
                    |                             |
                    | GPIO12 --------------------> RGB LED Red  
                    | GPIO13 --------------------> RGB LED Green  
                    | GPIO14 --------------------> RGB LED Blue  
                    +-----------------------------+
